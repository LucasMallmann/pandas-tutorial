Estudo sobre DataScience.

Este projeto inclui a análise do preço de casas nos EUA, e como elas se comportam ao longo dos anos.

Este projeto foi criado para estudos de DataScience utilizando Python, incluindo suas principais bibliotecas
de "Science" (ex: Pandas, Matplotlib, ScikitLearn)

Os dados foram extraídos do website Quandl, que oferece alguns dados abertos para análise, através de uma API.
